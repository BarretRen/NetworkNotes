# 基础概念

## cast 类型

- unicast 单播: 一对一
- multicast 组播: 一对一组, 节点需要监听对应的组播地址
- broadcast 广播: 一对广播域内所有节点, 节点需要处理. **广播域终结于路由器, 路由器不会转发**.

## 拓扑

- 物理拓扑: 实际的网络连接图
- 逻辑拓扑: 省略了 L2/L1 网络设备之后的纯三层网络连接图
- 节点 node: 网卡, 用 address 地址唯一标识
  - 三层地址: IP 地址, 整个 internet 范围内标识一个节点
  - 二层地址: 以太网 mac 地址, 用于在一个网络内标识一个节点

## hub 集线器工作原理

- L1 物理层设备
- 带宽共享
- 半双工模式: 同一时间只能发或收
- 只会以 flooding 方式发送数据
- CSMA/CD: 载波侦听多路访问/冲突检测

## switch 交换机工作原理

- 交换机是 L2 数据链路层设备, 能理解 2 层 mac 地址
- 自动学习 mac 地址表(mac 地址, port 口, vlan)
- 可以精确转发
